使用#

参考文档，先这样，然后这样，再那样，就好啦～

# 注意：由于Dart不支持反射等操作，导致dao及相关类不能自动生成，需要手动执行以下指令进行生成。
$ dart run build_runner build

问题#

  1. Tables can’t override primaryKey and use autoIncrement()
    设置自增就默认为主键，不用再额外指定主键了，但是指定了也只是警告，问题不大。
